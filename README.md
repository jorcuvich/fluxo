# Lógica Flow

Simulador educacional de fluxogramas para o ensino de lógica de programação e Pensamento Computacional. Desenvolvido com HTML, CSS e JavaScript puros, sem dependências externas, rodando inteiramente no navegador — sem necessidade de instalação.

**Acesse a versão online:** [conicas.com.br/fluxo](https://conicas.com.br/fluxo/)

---

## O que é o Lógica Flow

O Lógica Flow permite construir algoritmos por meio de blocos de fluxograma (arrastar e soltar), conectá-los visualmente e executá-los passo a passo, observando em tempo real o comportamento do fluxo de controle e o estado das variáveis na memória.

O objetivo é tornar visível e manipulável algo que, no ensino tradicional de programação, costuma ficar abstrato: o que acontece "por dentro" de um algoritmo enquanto ele roda.

## Funcionalidades

- **Blocos de construção por arrastar e soltar**: início, fim, processo, decisão, entrada (leia), saída (escreva) e pontos de desvio para organizar visualmente o fluxo.
- **Execução passo a passo ou automática**, com controle de velocidade (1x, 2x, 10x, máxima).
- **Painel de Memória RAM**, mostrando os valores das variáveis em tempo real durante a execução.
- **Console de Execução**, exibindo as saídas do programa conforme ele roda.
- **Detector de loop infinito**, interrompendo a execução automaticamente após 2000 passos.
- **Galeria de templates prontos** (Somador Simples, Maior ou Menor, Contador com Repetição), para começar rapidamente ou usar como exemplo em sala de aula.
- **Exportação e importação de projetos em JSON**, permitindo salvar e reabrir fluxogramas.
- **Guia de uso integrado** ("Como Jogar o Lógica Flow"), com explicação de cada bloco em linguagem simples, pensado para estudantes iniciantes.
- **Tema claro/escuro** e **layout responsivo**, com suporte a dispositivos móveis.

## Como usar

1. Acesse [conicas.com.br/fluxo](https://conicas.com.br/fluxo/) diretamente no navegador — não é necessário instalar nada.
2. Clique em **❔ Como Usar (Guia)** para uma introdução rápida aos blocos e à lógica de construção.
3. Arraste os blocos do menu lateral para a área de trabalho e conecte-os pelas portas azuis.
4. Use **▶ Executar Tudo** para rodar o algoritmo completo, ou **Passo a Passo ➡** para acompanhar a execução em detalhe, observando o painel de Memória RAM e o Console.
5. Para começar mais rápido, escolha um exemplo pronto em **📚 Galeria de Projetos** e clique em **Carregar Modelo**.

## Contexto de aplicação

O Lógica Flow já foi aplicado em contexto real de ensino, em turma de Ensino Técnico Integrado em Desenvolvimento de Sistemas, na atividade de implementação da fórmula de Bhaskara. A aplicação permitiu observar, entre outros aspectos, padrões de erro conceitual dos estudantes durante a construção do algoritmo — evidência que vem orientando o refinamento contínuo da ferramenta.

## Tecnologia

- HTML, CSS e JavaScript puros — sem frameworks ou dependências externas.
